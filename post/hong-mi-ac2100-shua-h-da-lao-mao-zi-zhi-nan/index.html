红米AC2100刷H大老毛子指南

2020-09-04

算是大白话，因为我也算是依葫芦画瓢，但是恩山现在的教程都太麻烦了，我还是单独写一个吧。

1.首先是前期准备

·蓝奏 ：https://ww.lanzous.com/icbb3wj

·老毛子固件下载 ：https://opt.cn2qq.com/padavan/RM2100_3.4.3.9-099.trx

·解压后启动 “npcap-0.9991” 安装npcap

·安装 putty ：https://www.putty.org/

·在开始菜单-设置-程序-程序和功能-启用和关闭Windows功能-勾选telnet client（注:不需要telnet server，现在win10也没有telnet服务器端了）

重要：禁用除连接路由那张网卡外其余所有接口，关闭防火墙和杀毒软件（如有）

2.把路由器的屁股像这样用网线连接好

3.设置电脑的IP

4.进入路由器开始设置

第一步

第二步

重要 ： 路由器的局域网IP地址需要修改，设置为：192.168.31.1

5.打开下载的文件夹rm2100，开启telnet命令.txt并复制里面的内容。这个我们马上就会使用

6.打开“一键开启telnet.bat”（不要用管理员运行！）

7.在新弹出的窗口有代码后就回到主窗口输入“Y”

8.弹出第三个窗口，出现图中第二行代码时马上粘贴我们早就准备好的代码，下方进度条100%之后我们进行下一步。

9.快捷键win+R，输入cmd打开命令指示符窗口，输入以下命令，完成后如下图
之后就可以刷机了。

复制粘贴到cmd里就行了：

telnet 192.168.31.1
wget http://192.168.31.177:8081/r3g.bin&&nvram set uart_en=1&&nvram set bootdelay=5&&nvram set flag_try_sys1_failed=1&&nvram commit
mtd -r write r3g.bin kernel1

这样就成功了

10.再次修改电脑IP

11.使用putty，winscp等工具

ip为：192.168.1.1
账号为：root或admin（自己试，我是root）
密码为:admin

进入winscp，把老毛子固件放到tmp目录下，改名字为redmi.trx（需要注意的是，winscp连接路由器的时候协议选scp）

进入putty，执行：

mtd -r write /tmp/redmi.trx kernel
