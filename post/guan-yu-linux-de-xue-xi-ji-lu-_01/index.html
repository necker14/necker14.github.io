关于Linux的学习记录_01

2020-02-03

这是Kevin的计算机学习博客记录

本次目标：

  1. 建立两个用户组group1和group2，以及三个用户dennis、daniel、abigale，并且将前2个用户分配在group1用户组下，后一个分配在group2用户组下。
  2. 创建一个whzy102.txt文件 将该文件的存取属性设置为对所有用户可读，可写和不可执行。
  3. 查找/bin目录中所有字母v开头的文件,且把定位结果重定向到文件wang2中。

进行操作的环境为：
·Windows 10 专业版
·VMware Workstation Pro 15.5正式版
·centos6.5正式发行版

第一项：
实现第一项我们可能用到的Linux指令有：groupadd、useradd、passwd

·首先，我们先建立两个用户组group1和group2

指令为：
groupadd group1
groupadd group2

·接着，我们将创建三个用户dennis、daniel、abigale，并按要求将前2个用户分配在group1用户组下，后一个分配在group2用户组下

指令为:
useradd -g group1 dennis
useradd -g group1 daniel
useradd -g group2 abigale

P.S. 指令说明:
groupadd 组名 创建一个用户组
useradd -g 组名 用户名 在一个现有组里添加一个用户

·但是我们不知道我们是否成功了，输入完指令Linux系统也没有任何的提示，我们可以使用id指令来检查我们的用户，语法为 id 用户名

指令为：
id dennis
id daniel
id abigale

至此，第一项就完成啦

第二项：
实现第二项我们可能用到的Linux指令有：touch、chmod

·首先，我们先创建一个whzy102.txt文件，使用ll指令查看创建是否成功

指令：
touch whzy102.txt
ll

·然后对whzy102.txt文件进行权限设置,存取属性设置为对所有用户可读，可写和不可执行,用ll指令检查是否修改成功

指令：
chmod a=rw whzy102.txt

P.S. 指令说明:
touch 文件名 在当前目录下创建一个文件
chmod a=rw 文件名 为XX文件开放所有人的读写权限

至此，第二项就完成啦

第三项：
实现第三项我们可能用到的Linux指令有：find、|、>

·首先，我们使用find指令查找/bin目录下的所有V开头文件

指令：
find /bin/ -name "v*"

将输出重定向到wang2中，并通过vim检查是否成功
指令：
find /bin/ -name "v*" >wang2.txt
vim wang2.txt

P.S.指令说明：
find /bin/ -name "v*" 在/bin目录下通过文件名寻找所有小写v开头的文件
find /bin/ -name "v*" >wang2.txt 将上指令的列表结果输出重定向到wang2.txt文件当中
vim wang2.txt 通过vim编辑器编写wang2.txt文件

至此，第三项就完成啦
